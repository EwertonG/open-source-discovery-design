# BabyBuddy Open Source Study

Projeto desenvolvido para a disciplina de Engenharia de Software I utilizando o projeto open source **BabyBuddy** como estudo de caso.

## Objetivo

Este trabalho documenta a análise, modelagem e evolução de um software livre sob a ótica da Engenharia de Software. Como contribuição prática, foi enviado um Pull Request que resolve o problema de dessincronização de telas, atualizando os detalhes de um timer quando encerrado por outra sessão ativa.
