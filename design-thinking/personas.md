# Personas

## Persona 1 – Ana Silva

**Idade:** 31 anos

**Profissão:** Engenheira de Software

### Perfil

Ana é mãe de primeira viagem e divide os cuidados do bebê com seu marido. Durante o dia, utiliza o BabyBuddy tanto no notebook quanto no celular para registrar alimentação, sono e outras atividades.

### Objetivos

- Registrar rapidamente as atividades do bebê.
- Compartilhar as informações com outros cuidadores.
- Garantir que os registros estejam sempre atualizados.

### Dores

- Alterna frequentemente entre dispositivos.
- Pode visualizar informações desatualizadas.
- Deseja confiar no estado apresentado pela aplicação.

### Necessidades

- Interface sincronizada entre diferentes sessões.
- Informações consistentes.
- Feedback claro quando alguma atividade for encerrada.

## Persona 2 – Carlos Oliveira

**Idade:** 36 anos

**Profissão:** Médico

### Perfil

Carlos possui uma rotina corrida e utiliza o BabyBuddy para registrar os cuidados realizados durante o dia. Muitas vezes inicia um registro pelo celular e continua utilizando o sistema pelo computador.

### Objetivos

- Registrar atividades de forma rápida.
- Evitar erros durante o acompanhamento do bebê.
- Utilizar o sistema em diferentes dispositivos.

### Dores

- Perder tempo corrigindo registros.
- Executar ações em timers que já foram encerrados.
- Não perceber alterações realizadas em outra sessão.

### Necessidades

- Sincronização das informações.
- Interface intuitiva.
- Mensagens claras sobre o estado dos timers.
